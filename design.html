Security Tutorial

This project will educate users on how to secure their smart contracts

Features

  • make a smart contract in browser(remix)
  • learn about security flaws by example
  • learn security fundamentals
  • pay for a contract security check

Pages

  • security
  • smart contracts
  • frontend

User Flows

  • anonymous user can click on the left sidebar to learn about fundamentals and case studies
  • anonymous user can play around with smart contracts in the browser using remix
